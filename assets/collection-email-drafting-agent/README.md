# Collection Email Drafting Agent

An AI agent that helps AR collections specialists draft personalized collection emails using overdue receivables, dunning history, and dispute context from SAP S/4HANA

## Overview

Uses A2A Protocol, LangGraph, LiteLLM, and SAP Cloud SDK.

## Structure

- `app/main.py` - A2A server entry
- `app/agent_executor.py` - Request handling
- `app/agent.py` - Agent logic
